Find the list of supported attributes for the <area> tag

Besides the global HTML attributes, the <area> tag supports the following HTML5 attributes: alt, coords, href, hreflang, media, rel, shape, target, and type.

Observations when comparing the W3C cheatsheet and the MDN attribute reference list:

  • The MDN attribute reference list shows also ‘non-HTML5 attributes’ and the attributes are listed in alphabetical order (this might be helpful), while the W3C cheatsheet shows only HTML5 attributes and does not list the supported attributes in a ‘logically’ order (at least to me).
  • The W3C cheatsheet does not list the download attribute in the initial enumeration of the supported HTML5 attributes for the <area> tag; if you follow the ‘Specification’ link the download attribute is listed, but there the media attribute is missing.

Does an attribute called 'src' exist? What is the purpose of this attribute and what are the elements it can be applied to?

Yes. It defines the URL of embeddable content and can be applied to the following elements: audio, embed, iframe, img, input, script, source, track, video.
